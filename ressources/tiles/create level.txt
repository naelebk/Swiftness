bonne pratique de création de niveau :

crée tous ces layers dans cet ordre :
tile layer -> [t]
object layer -> [o]

[t]> IO
[t]> Border
[t]> Collision_v
[t]> Collision_h
[t]> Collision_bloc
[o]> Gravity_switch
[t]> Filling
[t]> Decoration 

---

lorsqu'une plateforme verticale rejoint une plateforme horizontale dans un angle, toujours privilégié la plateforme verticale

n'utiliser le tileset utilities que dans des tile invisible (Border et IO)
